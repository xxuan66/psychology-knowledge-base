## 🧠 自我接纳的真相

### 自我接纳是什么？

**自我接纳 = 如实地看见自己 + 无条件地爱自己**

**不是：**
- ❌ 放弃成长
- ❌ 容忍坏习惯
- ❌ "我就这样，爱咋咋地"

**是：**
- ✅ 承认自己的优点和缺点
- ✅ 不因缺点而否定自己
- ✅ 从爱自己的角度改变

---

### 为什么自我接纳很难？

**原因 1：内在批评家**

每个人内心都有个批评家：
- "你真笨"
- "这都做不好"
- "别人会笑话你"

这个声音可能来自：
- 童年时父母/老师的评价
- 过去的失败经历
- 社会比较

---

**原因 2：条件化的爱**

从小被教育：
- "考第一名才是好孩子"
- "要听话才有人喜欢"
- "要优秀才值得被爱"

**结果：** 认为爱是有条件的，必须优秀才配被爱。

---

**原因 3：社会比较**

社交媒体时代：
- 看到别人的高光时刻
- 用自己的幕后对比别人的台前
- 永远有人比你"更好"

## 🚀 自我接纳练习

### 练习 1：识别内在批评家

**步骤：**

```
1. 留意内心的声音
2. 记录批评的内容
3. 问自己："这是谁的声音？"
4. 挑战它："这是真的吗？"
```

**示例：**

```
批评："你真笨，这都做不好"

挑战：
- 这是真的吗？还是想法？
- 我会对好朋友说这样的话吗？
- 有什么证据证明这是真的？
- 有什么证据证明这不是真的？
```

---

### 练习 2：自我关怀冥想

**步骤：**

```
1. 找个舒服的姿势坐下或躺下
2. 把手放在胸口
3. 对自己说：

"此刻我正在经历痛苦"（承认痛苦）
"痛苦是人生的一部分，我不是一个人"（共同人性）
"愿我对自己温柔一些"（自我关怀）

4. 重复几次，感受手心的温暖
```

---

### 练习 3：写给自己的信

```
亲爱的____：

我知道你经常批评自己，
觉得自己不够好，
不够聪明，不够漂亮，不够成功。

我想告诉你：

你不需要完美才值得被爱。
你的存在本身就有价值。
你已经在尽力了。

从今天开始，
我会用对待好朋友的方式对待你。
温柔、理解、支持。

爱你的，
____
```

---

### 练习 4：优点清单

**步骤：**

```
1. 列出你的 10 个优点
2. 不管多小都可以
3. 每天看一遍
4. 持续添加
```

**示例：**
- 善良
- 有同理心
- 做事认真
- 会做饭
- 坚持运动
- 善于倾听
- ...

**关键：** 真诚地认可自己。
