## 🧠 抗压能力的真相

### 什么是抗压能力？

**抗压能力 = 面对压力时的适应和恢复能力**

**不是：**
- ❌ 没有情绪波动
- ❌ 从不感到压力
- ❌ 硬扛不休息

**是：**
- ✅ 能够感受情绪
- ✅ 能够有效应对
- ✅ 能够从中成长

---

| 要素 | 含义 | 如何培养 |

| **情绪觉察** | 识别自己的情绪 | 身体扫描、命名情绪 |

## 🚀 建立心理韧性

### 方法 1：认知重构

**核心：** 改变对压力的看法

**从：**
"压力是坏事，会毁掉我"

**到：**
"压力是成长的机会，让我更强"

---

**练习：重新定义压力**

```
压力情境 → 重新定义

公开演讲 → 展示自己、帮助他人的机会
工作挑战 → 提升能力、证明自己的机会
人际冲突 → 学习沟通、了解他人的机会
```

---

### 方法 2：累积小成功

**核心：** 自信来自一次次"我做到了"

**步骤：**

```
1. 设定一个极小的目标
2. 立即去做
3. 完成后认可自己
4. 继续下一个
```

**示例：**
- 整理床铺
- 运动 5 分钟
- 读 10 页书
- 完成一项工作

---

### 方法 3：建立支持系统

**核心：** 没有人能独自坚强

**支持系统包括：**

```
→ 家人朋友（情感支持）
→ 同事伙伴（专业支持）
→ 专业人士（心理咨询）
→ 社群组织（归属感）
```

**练习：** 列出 3 个可以求助的人。

---

### 方法 4：自我关怀

**核心：** 像对待朋友一样对待自己

**练习：**

```
当遇到困难时，对自己说：

"这确实很难"（承认困难）
"很多人都会遇到这样的情况"（共同人性）
"愿我对自己温柔一些"（自我关怀）
```

## 💪 压力转化技巧

### 技巧 1：压力重评

**步骤：**

```
1. 当感到压力时，停下来
2. 问自己："这个压力想告诉我什么？"
3. 找到积极意义
4. 转化为行动动力
```

**示例：**

```
压力：明天要演讲，很紧张

重评：
→ 紧张说明我在乎
→ 紧张让我更专注
→ 紧张是能量，可以利用
```

---

### 技巧 2：成长思维

**固定思维 vs 成长思维：**

| 固定思维 | 成长思维 |
|---------|---------|
| "我不擅长这个" | "我还不擅长，但可以学" |
| "失败=我很差" | "失败=学习机会" |
| "挑战=威胁" | "挑战=成长" |
| "别人成功=我不行" | "别人成功=我可以学习" |

---

### 技巧 3：感恩练习

**核心：** 感恩能提升心理韧性

**练习：**

```
每天睡前记录 3 件感恩的事：

→ 今天阳光很好
→ 同事帮我解决了问题
→ 家人做了我爱吃的菜
```

**关键：** 专注于当时的感受。
