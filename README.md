# 心理学知识库 (Psychology Knowledge Base)

🧠 系统化的心理学知识体系，从基础理论到实践应用

## 📚 内容架构

```
psychology-knowledge-base/
├── docs/           # 文档资料
├── content/        # 核心内容
│   ├── 基础理论/   # 心理学基础概念、理论框架
│   ├── 实践应用/   # 实际应用场景、方法技巧
│   ├── 案例分析/   # 真实案例、深度解析
│   └── 工具模板/   # 测评工具、练习模板
├── scripts/        # 自动化脚本
├── logs/           # 运营日志
└── archive/        # 归档内容
```

## 🎯 内容方向

- **基础理论**：心理学核心概念、主要流派、经典理论
- **实践应用**：自我认知、情绪管理、人际关系、成长方法
- **案例分析**：真实案例解析、常见问题解答
- **工具模板**：心理测评、练习表格、自助工具

## 📝 内容输出

内容同步发布至：
- 飞书文档（知识库沉淀）
- 微信公众号（内容传播）

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/xxuan66/psychology-knowledge-base.git

# 进入目录
cd psychology-knowledge-base

# 查看内容结构
tree -L 2
```
