## 🧠 失恋的真相

### 失恋为什么这么痛？

**生理层面：**

恋爱时大脑分泌：
- 多巴胺（快乐）
- 催产素（依恋）
- 血清素（稳定）

分手后这些激素骤降，**类似戒毒反应**。

---

**心理层面：**

- 失去依恋对象
- 自我价值感崩塌
- 未来规划破灭
- 生活习惯被打乱

---

### 失恋的 5 个阶段

**库伯勒 - 罗丝模型：**

```
1. 否认："这不是真的"
2. 愤怒："TA 怎么能这样对我"
3. 讨价还价："如果当初我...就好了"
4. 抑郁："没有 TA 我活不下去"
5. 接受："我好了，可以向前看了"
```

**这些阶段不是线性的，会反复。**

## 🚀 走出失恋的方法

### 方法 1：允许悲伤

**不要压抑情绪。**

```
→ 想哭就哭
→ 想发泄就发泄
→ 给自己时间悲伤
```

**压抑只会延长痛苦。**

---

### 方法 2：断联

**步骤：**

```
1. 删除/拉黑联系方式
2. 不看 TA 的动态
3. 不回复 TA 的消息
4. 至少坚持 30 天
```

**为什么：**
- 每次联系都是重新撕开伤口
- 给大脑时间"戒毒"
- 让自己真正放下

---

### 方法 3：书写疗愈

**练习：**

```
1. 给 TA 写一封信（不发送）
2. 写下所有想说的话
3. 写下你的感受
4. 写完后烧掉或撕掉
```

**好处：** 释放情绪，获得closure。

---

### 方法 4：重新定义故事

**从：**
"TA 抛弃了我，我不够好"

**到：**
"这段关系结束了，我学到了____，我值得更好的"

---

### 方法 5：重新爱上自己

**步骤：**

```
1. 列出你喜欢自己的 10 个点
2. 每天做一件让自己开心的事
3. 投资自己（学习、运动、旅行）
4. 重建社交圈
```
