## 🧠 孩子行为的真相

### 行为是冰山一角

```
可见：行为（发脾气、不听话）
      ↓
不可见：感受（愤怒、伤心、害怕）
      ↓
需求（被理解、被关注、自主权）
```

**示例：**

```
行为：孩子不写作业

可能的需求：
- 太难了，需要帮助
- 太累了，需要休息
- 想玩，需要自主权
- 害怕失败，需要鼓励
```

---

### 孩子的 4 个核心需求

| 需求 | 表现 | 满足方式 |
|------|------|---------|
| **安全感** | 黏人、分离焦虑 | 稳定的爱、规律作息 |
| **被理解** | 发脾气、哭闹 | 倾听、共情 |
| **自主权** | 叛逆、说不 | 给选择、尊重意见 |
| **成就感** | 退缩、放弃 | 鼓励、小步骤成功 |

## 🚀 有效沟通方法

### 方法 1：蹲下来沟通

**步骤：**

```
1. 蹲到孩子的高度
2. 眼神接触
3. 用平静的语气
4. 简短清晰的指令
```

**为什么有效：**
- 减少压迫感
- 孩子感到被尊重
- 更容易听进去

---

### 方法 2：共情回应

**公式：** 命名情绪 + 理解感受

**示例：**

```
孩子："我不想上学！"

❌ "不上学怎么行！必须去！"

✅ "听起来你今天不太想去学校。
   发生什么事了吗？"
```

---

### 方法 3：给选择，不下命令

**示例：**

```
❌ "马上收拾玩具！"

✅ "你想先收拾积木，还是先收拾汽车？"

❌ "快去洗澡！"

✅ "你想现在洗澡，还是 5 分钟后洗澡？"
```

**好处：** 孩子感到有自主权，更愿意合作。

---

### 方法 4：我信息表达

**公式：** 行为 + 感受 + 影响

**示例：**

```
❌ "你怎么这么不听话！"

✅ "玩具撒在地上（行为），
   妈妈很担心有人会绊倒（感受），
   我们一起收拾好吗？（请求）"
```

## 💪 设定边界

### 温和而坚定

**步骤：**

```
1. 提前说清楚规则
2. 违反时，平静地执行后果
3. 不妥协，但不发火
```

**示例：**

```
规则："每天只能看 30 分钟动画片"

违反时：
"时间到了，要关电视了。"
（孩子哭闹）
"我知道你还想看，但规则是 30 分钟。
明天还可以继续看。"
（平静地关掉）
```

**关键：** 温和（不发火）+ 坚定（不妥协）

---

### 自然后果法

**让孩子体验行为的自然后果。**

**示例：**

```
→ 不吃饭 → 会饿（不提供零食）
→ 不收拾玩具 → 找不到（不帮忙找）
→ 拖延作业 → 被老师批评（不代劳）
```

**好处：** 孩子学会为自己的行为负责。
