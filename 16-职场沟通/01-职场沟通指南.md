## 🚀 向上沟通

### 和领导沟通的原则

**1. 结果导向**

领导关心的是结果，不是过程。

```
❌ "我做了很多努力，但是..."

✅ "目前进展是____，预计____完成，
   遇到的问题是____，需要您的支持是____"
```

---

**2. 带着方案提问**

```
❌ "领导，这个怎么办？"

✅ "领导，这个问题我有两个方案：
   A 方案____，优点是____，缺点是____
   B 方案____，优点是____，缺点是____
   我建议 A 方案，您看呢？"
```

---

**3. 及时反馈**

```
→ 任务开始后：确认理解
→ 进行中：定期汇报进展
→ 遇到问题：及时求助
→ 完成后：总结复盘
```

---

### 如何优雅地拒绝领导

**公式：** 理解 + 现状 + 建议

**示例：**

```
领导："这个任务今天能完成吗？"

❌ "做不完，太多了。"

✅ "我理解这个任务很紧急（理解）。
   目前我手上有 A、B 两个任务也在今天截止（现状）。
   您看是否可以优先完成这个，A、B 任务延后到明天？（建议）"
```

## 🤝 平级沟通

### 请求同事帮助

**公式：** 背景 + 请求 + 价值

**示例：**

```
"我现在在做一个项目（背景），
需要你在____方面支持一下（请求），
这个项目完成后对咱们团队____有帮助（价值）。
你看方便吗？"
```

---

### 应对甩锅

**原则：** 对事不对人，用事实说话

**示例：**

```
同事："这个数据错了，是你的问题吧？"

❌ "怎么可能！明明是你给我的原始数据就有问题！"

✅ "我理解这个数据错误会影响项目。
   我们一起查一下问题出在哪里？
   我记得原始数据是从你那边导出的，
   我们一起核对一下源头？"
```

---

### 优雅地说"不"

**公式：** 感谢 + 拒绝 + 替代方案

**示例：**

```
同事："能帮我做一下这个吗？"

❌ "不行，我很忙。"

✅ "谢谢你信任我（感谢）。
   但我现在手上有几个紧急任务，实在抽不出时间（拒绝）。
   你可以试试____，或者等下周我有空再帮你（替代）。"
```

## ⚔️ 冲突处理

### 职场冲突的 3 个原则

**1. 私下沟通**

不在公开场合争执，给对方面子。

---

**2. 对事不对人**

```
❌ "你这个人怎么这样"

✅ "这个做法有问题"
```

---

**3. 聚焦解决**

```
❌ "都是你的错"

✅ "我们一起想想怎么解决"
```

---

### 冲突沟通步骤

```
1. 冷静后主动沟通
2. 描述事实，不评判
3. 表达感受，不指责
4. 提出建议，不命令
5. 寻求共识，不争输赢
```
