## 📦 内容组织方式

### 当前架构（按主题分类）
- **01-情绪管理** - 焦虑、抑郁、愤怒等情绪问题
- **02-习惯改变** - 拖延、时间管理、习惯养成
- **03-亲密关系** - 沟通、爱情、婚姻、家庭
- **基础心理学** - 理论框架、概念解析
- **发展心理学** - 人生各阶段心理发展

### 备用架构（标准 content 目录）
- **基础理论** - 心理学核心概念
- **实践应用** - 实际场景应用
- **案例分析** - 真实案例解析
- **工具模板** - 测评工具、练习表格

## 📝 内容导出流程

### 导出到飞书文档
1. 选择要导出的 markdown 文件
2. 使用飞书文档管理器发布
3. 自动渲染格式和表格
4. 设置权限和分享链接

### 导出到公众号
1. 选择要发布的文章
2. 使用公众号运营工具排版
3. 添加封面图和摘要
4. 定时发布或立即发布

---

## 🔄 同步策略

### GitHub → 飞书/公众号
- **单向同步**：GitHub 作为内容源
- **版本控制**：GitHub 保留所有历史版本
- **内容沉淀**：飞书作为知识库归档
- **内容传播**：公众号作为发布渠道

### 更新流程
1. 在 GitHub 编辑/新增内容
2. Commit 并 Push 到仓库
3. 定期批量导出到飞书/公众号
4. 保持内容一致性
