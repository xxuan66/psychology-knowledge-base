## 🚀 目标设定法

### SMART 原则

**好的目标应该符合 SMART 原则：**

| 原则 | 含义 | 示例 |
|------|------|------|
| **S 具体** | 明确清晰 | ❌ "我要变健康" → ✅ "我要每周运动 3 次" |
| **M 可衡量** | 有量化标准 | ❌ "多读书" → ✅ "每月读 2 本书" |
| **A 可实现** | 有挑战但可行 | ❌ "一年赚 1000 万" → ✅ "年收入增长 20%" |
| **R 相关** | 与价值观一致 | 问自己："这对我真的重要吗？" |
| **T 有时限** | 有截止日期 | ❌ "总有一天" → ✅ "6 月 30 日前" |

---

### 目标拆解

**大目标→中目标→小目标→行动**

**示例：**

```
大目标：一年内升职

中目标：
- Q1: 完成 X 项目
- Q2: 提升 Y 技能
- Q3: 建立 Z 成果

小目标（本月）：
- 完成项目第一阶段
- 学习 Y 技能基础课程
- 和领导沟通一次职业发展

行动（今天）：
- 制定项目计划
- 学习 30 分钟
- 预约领导时间
```

## 💪 保持动力

### 方法 1：可视化成功

**步骤：**

```
1. 闭上眼睛
2. 想象目标达成后的场景
3. 感受当时的喜悦和自豪
4. 记住这种感觉
```

**每天练习，增强动力。**

---

### 方法 2：记录进步

**方式：**

```
→ 写日记
→ 拍照记录
→ 数据追踪
→ 打卡社群
```

**看到进步，更有动力继续。**

---

### 方法 3：设置奖励

**示例：**

```
→ 完成小目标：看一集喜欢的剧
→ 完成中目标：吃一顿大餐
→ 完成大目标：买想要的东西/旅行
```

**奖励要及时、愉悦、可持续。**

---

### 方法 4：找伙伴

**方式：**

```
→ 找志同道合的人
→ 定期汇报进展
→ 互相鼓励监督
→ 加入社群
```

**有人同行，更容易坚持。**

---

## ⚠️ 应对挫折

### 挫折是必然的

**预期会遇到困难：**

```
→ 动力下降
→ 遇到瓶颈
→ 生活变故
→ 自我怀疑
```

**这些都是正常的，不是失败。**

---

### 应对策略

**策略 1：调整目标**

```
目标太大？→ 缩小
时间太紧？→ 延长
方法不对？→ 调整
```

---

**策略 2：休息充电**

```
→ 允许自己暂停
→ 做喜欢的事
→ 恢复精力
→ 再出发
```

---

**策略 3：重新连接"为什么"**

```
问自己：
- 我为什么要设定这个目标？
- 这对我真的重要吗？
- 达成后会有什么改变？
```

---

**策略 4：原谅自己**

```
❌ "我又失败了，我真没用"

✅ "这次没做好，我学到了____。
   明天重新开始。"
```
